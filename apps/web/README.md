# DataDock Web

This package contains the DataDock Next.js web application.

Repository root:

```text
../../
```

Use the root workspace commands:

```bash
pnpm dev
pnpm build
pnpm lint
pnpm typecheck
```

Do not install dependencies from this folder directly. Run `pnpm install` from the repository root.
